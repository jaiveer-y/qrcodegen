
# Qr Code Generator

This converts an input into a qr code and save it as a png. 

## Demo

![demo](https://github.com/jaiveer-y/qrcodegen/blob/master/code.png?raw=true)


## Installation



```bash
    gh repo clone jaiveer-y/qrcodegen
    
    pip install pyqrcode
    pip install pypng
    pip install pillow
    pip install pyzbar
```
